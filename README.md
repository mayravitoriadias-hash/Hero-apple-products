**Apple Products Hero Slider (HTML, CSS & JS)**

This project is a hero section for a landing page featuring an automatic sliding showcase of Apple products.

It was built as a learning exercise to practice layout creation, visual hierarchy, and basic animation.

What’s included
Hero section design focused on product presentation
Automatic sliding effect 
Structured HTML and styled with CSS

**Purpose**

This project is part of my front-end learning journey.
It helped me practice:

Building hero sections for landing pages
Creating simple animations and transitions
Working with layout and positioning
Improving visual hierarchy and product emphasis

**Note**

This is not a production-ready project.
It is a practice project focused on HTML and CSS fundamentals, especially in creating engaging hero sections.
